    Personal Intiatives

    This section showcases a collection of my independent initiatives, reflecting my passion for experimenting with cutting-edge technologies and creative problem-solving. Explore these projects to see how I bring innovation and a personal touch to my work.

    Chatbot using Llama 2

    Objective: Develop a chatbot using the Llama 2 language model powered by Replicate to interact with users in real-time and provide effective responses to queries.
    Approach: Utilized Streamlit to create a user-friendly web interface, integrated the Llama 2 model, and handled API tokens securely as environment variables.
    Skills Utilized: Streamlit, Replicate, Python, Llama 2 Language Models.
    Result/Outcome: Created an efficient chatbot capable of assisting users with real-time responses.

    Chatbot using FLAN-T5-XL and Streamlit

    Objective: Create a chatbot using the FLAN-T5-XL model from Hugging Face's Transformers library to engage in conversations with users based on predefined prompts.
    Approach: Implemented the chatbot's backend logic using Python, utilized PyTorch and Transformers library for model handling, and developed the web interface using Streamlit.
    Skills Utilized: Python, PyTorch, Transformers, Streamlit.
    Result/Outcome: Developed an interactive chatbot accessible via a web browser.

    Chat with the Custom Docs using LlamaIndex

    Objective: Build a chatbot capable of providing technical answers about the Streamlit Python library by leveraging the LlamaIndex and OpenAI's GPT-3.5-turbo.
    Approach: Employed Streamlit for web interface creation, utilized LlamaIndex for efficient document indexing and search, and integrated OpenAI's GPT-3.5-turbo for generating responses.
    Skills Utilized: Streamlit, LlamaIndex, OpenAI's GPT-3.5-turbo, Python.
    Result/Outcome: Enabled users to obtain accurate and technical answers based on indexed documentation.

    Blog Generator using Falcon-7B

    Objective: Develop a Streamlit app for generating blog content using the Falcon-7B Instruct language model.
    Approach: Users provide a topic, and the app generates a blog with an introduction, explanation, and summary using the Falcon-7B Instruct model.
    Skills Utilized: Streamlit, Falcon-7B Instruct model, Python.
    Result/Outcome: Created a user-friendly app for effortless blog content generation based on user-provided topics.

    Askbuddy chatbot using Hugchat & streamlit

    Objective: Create a user-friendly chatbot for real-time interactions, providing effective responses.
    Approach: Utilized Streamlit, integrated HugChat for AI responses.
    Skills Utilized: Streamlit, HugChat, Pillow, Python.
    Result/Outcome: An efficient chatbot enhancing user engagement and interaction.

    Chat with CSV file

    Objective: Create an interactive tool that enables users to query data from CSV files using natural language, powered by the GPT-3.5 Turbo language model, to facilitate data exploration and analysis.
    Approach to Solve:
    Streamlit Integration: Developed a user-friendly web interface using Streamlit for data interaction.
    Langchain and GPT-3.5 Turbo: Utilized Langchain and the GPT-3.5 Turbo language model from OpenAI to generate responses to data queries.
    Secure API Key Handling: Implemented a secure method for users to input their OpenAI API key.
    Skills Utilized: Streamlit, Langchain, GPT-3.5 Turbo, Pandas, Python.
    Result/Outcome: Created a powerful tool that allows users to load and interact with CSV data using natural language queries. The integration of GPT-3.5 Turbo enhances the application's ability to provide meaningful responses, making it a valuable asset for data exploration and analysis.
